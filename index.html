Day 59 Task - 90 Days of DevOps

Welcome to Day 59 of the 90 Days of DevOps program! Today's task is to set up an Nginx server and host a static HTML page. We'll also clone content from a Git repository.

Task Overview

1. Install Nginx on your server.

2. Clone a Git repository containing your HTML page and assets.

3. Ensure Nginx is running and serving your HTML content.

Get Started

Follow the instructions in your Ansible playbook to complete the task. Make sure to check your server's IP and SSH key.
